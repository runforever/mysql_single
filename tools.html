MySQLTuner-perl

MySQLTuner-perl 可以对当前的运行的 MySQL 提供优化建议，对于新手来说帮助挺大的。

安装

git clone https://github.com/major/MySQLTuner-perl.git

使用

perl mysqltuner.pl --host targetDNS_IP --user admin_user --pass admin_password
